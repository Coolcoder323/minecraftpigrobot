# minecraftpigrobot

### Created by coolcoder823

This alpha-phase, open-source project provides files for an autonomous, Minecraft-inspired pig robot, combining Arduino, 3D printing, and robotics.

## Technical Documentation
The project utilizes an Arduino Giga, PCA9685 I2C driver, 4x PIR motion sensors, and 9g micro servos.
