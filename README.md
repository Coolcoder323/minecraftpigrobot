# minecraftpigrobot

Created by coolcoder823

This repository contains source code, schematics, and 3D printable files for an open-source, educational robot designed to resemble a Minecraft pig.

*Note: This project is in its alpha phase and is incomplete.*

## Technical Documentation

The hardware includes an Arduino Giga, 4x PIR motion sensors, 9g micro servos, and a PCA9685 I2C servo driver.
